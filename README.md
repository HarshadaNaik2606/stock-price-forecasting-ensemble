# Stock-price-forecasting-ensemble
Predictive modelling framework for stock price forecasting integrating Linear Regression, LSTM networks, and stacking ensembles using Decision Tree and XGBoost. Includes data preprocessing, feature engineering, hyperparameter tuning, model comparison, and performance evaluation using Apple Inc. (AAPL) stock data.
